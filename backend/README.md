# Online Learning Platform Backend

A Node.js/Express API for the Online Learning Platform.

## Installation

```bash
npm install
```

## Running

For development:
```bash
npm run dev
```

For production:
```bash
npm start
```

The server will run on port 3000 by default, or the PORT environment variable.

## API Endpoints

- GET /api/user - Get user information
- GET /api/courses - Get all courses
- GET /api/assignments - Get assignments
- GET /api/upcoming-deadlines - Get upcoming deadlines
- GET /api/recent-activity - Get recent activity
- GET /api/community-posts - Get community posts
- GET /api/exams - Get exams
